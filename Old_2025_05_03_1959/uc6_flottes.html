Selon vos attentes, la fréquence le type de données, le type de véhicules et la taille de votre flotte...

INTERCONNEXION EXTERNE PAR API

  • Ajouter les données à son logiciel : barème kilométrique, carte carburant, gestion de planning, géolocalisation, affectation de véhicules, diagnostic (entretien, pannes, code erreurs), données connexes (suivi de température, équipements du véhicule, détection de porte ouverte...)
  • Bénéficier d’un service 360 sur un seul outil
  • Gagner du temps
  • Choisir la simplicité
  • Ajouter de la valeur aux outils internes (métiers déjà intégrés)
  • Améliorer la communication entre les utilisateurs

GESTION DE TOUS LES TYPES DE VÉHICULES (VL, VUL, PL, thermique, électrique, hybride)

  • Géolocaliser tous types de véhicules : Véhicules utilitaires légers, Véhicules Particuliers, Poids lourds, Remorques, Triporteur, Le fret (palettes, conteneurs...)...
  • Suivre les données à distance
  • Personnaliser les alertes
  • Historiser les données
  • Générer des rapports
  • Cartographier les véhicules
  • Gérer les sinistres
    • Données remontées : Géolocalisation, suivi de trajets, codes erreur, diagnostics, entretiens, données d’écoconduite et connexes.

GESTION DES TRAJETS PRO/PERSO ET IDENTIFICATION CHAUFFEUR

  • Géolocaliser tous types de véhicules
  • Gérer les déclarations URSAFF (véhicules de fonction et de service)
  • Etablir des plages horaires pro perso (auto ou manuel)
  • Identifier un conducteur qui utilise un véhicule
  • Simplifier la gestion administrative
  • Faciliter l’accompagnement des conducteurs
  • Vérifier l’utilisation du véhicule
    • Données remontées : GPS, identifiant du conducteur, type de trajet.

ÉCOCONDUITE

  • Faire des économies : conso, gain de temps, entretien
  • Maitriser son empreinte environnementale
  • Mettre en place une politique RSE
  • Remontée des données d’accéléromètre,durée et distance sans consommation, régime moteur, type de route, régulateur, etc.
    • Données remontées : GPS, consommation, freinage, RPM, comportement de conduite...

ENREGISTREMENT NUMÉRIQUE DES DONNÉES SOCIALES PL/VUL

  • Géolocaliser les véhicules à distance
  • Suivre les trajets
  • Personnaliser les alertes
  • Historiser les données
  • Se conformer à la réglementation en vigueur
    • Données remontées : GPS, données sociales (conducteur et véhicule), chronotachygraphe numérique (PL et VUL).
